## Project Description
This project consists of creating two types of documents. The 1st one is a project scope statement written in Markdown and based on chapter 3 of our reading. Next is a simple, single-page personal website built using semantic HTML. The scope statement includes the project's purpose, expectations, and boundaries. At the same time, the HTML webpage provides a practical implementation of that scope by presenting personal and professional information and making it available as an online document.
